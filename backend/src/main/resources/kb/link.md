# Chainlink (LINK)

## Identity
Chainlink is a decentralized oracle network that connects blockchains to external (off-chain) data, computation, and systems. It launched its mainnet in 2019, founded by Sergey Nazarov and Steve Ellis under the company SmartContract / Chainlink Labs. Its token is LINK. Chainlink is not its own layer-1 blockchain; it is middleware that runs across many blockchains (Ethereum and others), where LINK is issued as an ERC-20-compatible token.

## Consensus
Chainlink does not have its own base-layer consensus like a PoW or PoS chain. Instead, it relies on decentralized oracle networks: independent node operators retrieve and report data, and their responses are aggregated (for example, off-chain reporting where nodes sign a single aggregated report submitted on-chain). Node operators stake or hold LINK and build reputations; data integrity comes from decentralization across many independent nodes and data sources, and from cryptoeconomic incentives including staking.

## Supply schedule
LINK has a fixed total supply of 1 billion tokens, all created at inception. There is no protocol inflation that mints new LINK beyond the original supply. Tokens were distributed across a public sale, the company, and reserves used to incentivize node operators and ecosystem growth, so circulating supply increases as reserves are released rather than through new issuance.

## Use case
LINK is used to pay node operators for delivering data and services, and is staked by node operators and stakers to back the security and reliability of oracle services. Chainlink's services include decentralized price feeds (widely used by DeFi protocols), verifiable randomness (VRF), automation (keepers), proof of reserve, and the Cross-Chain Interoperability Protocol (CCIP) for moving data and tokens between chains.

## Key risks
Risks include reliance on the honesty and availability of oracle node operators and underlying data sources (oracle manipulation or failure can affect dependent protocols); competition from other oracle providers; dependence on demand from smart-contract platforms; incentive and staking-design challenges; and general regulatory uncertainty for tokens.

## On-chain footprint
Because Chainlink operates across many chains, its footprint is measured by oracle activity rather than a single ledger: the number of active price feeds and data updates, the value secured by its oracles, the number of integrations and node operators, and the amount of LINK staked. CCIP message volume is also tracked.
